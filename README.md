# Student-Record-Manager
A Python programming for managing student records with full CRUD operations, built to demonstrate core Python programming skills.

## Features

- **Add Students**: Create new student records with details (name, age, grade, email, phone)
- **View Students**: List all students with formatted output
- **Search Students**: Search by name, age, or grade
- **Update Students**: Modify existing student information
- **Delete Students**: Remove student records
- **Statistics**: View analytics (total students, average age, grade distribution)
- **Data Persistence**: Automatic save/load from JSON file
- **Backup/Restore**: Create and restore from backup files
